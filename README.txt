Lab 5 - Music Classification

Implemented Features:
	Danceability
    RhythmDescriptors2013
    BeatsLoudness
    Rms
    BpmHistogramDescriptors
    Dynamiccomplexity

Usage:
	Modify from "main.py" the variables INPUT_PATH with the path containing the files from where we want to extract the features, OUTPUT_PATH_FILE with the name and path where we want to write the result csv file and comment (#) line feature from the FEATURES list that you DON'T want to extract (if you want you want to extract all the aviable features, let all the files uncommented).
